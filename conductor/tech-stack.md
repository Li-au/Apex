# Tech Stack - APEX

## Frontend
- **Framework:** React 18+
- **Styling:** Tailwind CSS
- **State Management:** React Context + hooks (useState, useReducer)
- **Build Tool:** Vite
- **Package Manager:** npm

## Backend/Services
- **None (100% client-side)** - Données sauvegardées localement (localStorage)
- **Optional Future:** Firebase pour leaderboards et cloud saves

## Monetization
- **Ads:** Google AdSense (web ads)
- **IAP:** Google Play Billing (pour Android, futur)
- **Analytics:** Google Analytics 4

## Deployment
- **Hosting:** GitHub Pages (gratuit, déploiement auto avec GitHub Actions)
- **Alternative:** Vercel (gratuit aussi)
- **Domain:** Domaine gratuit ou custom (optionnel)

## Development Tools
- **Version Control:** Git + GitHub
- **Code Quality:** ESLint, Prettier
- **Testing:** Jest + React Testing Library (optionnel pour MVP)
- **Performance:** Lighthouse, Chrome DevTools

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Performance Targets
- First Paint: < 1s
- Time to Interactive: < 2s
- Bundle size: < 500KB (gzipped)
